# Project Description
This project is a simple hospital appointment system built using Java Swing.
The main idea was to create a user-friendly system where a patient can book an appointment step by step.
The user first enters personal details, then selects a doctor and specialization, and finally confirms the appointment.
This project shows how a real-life hospital booking process can be turned into a basic digital system.

# Features
- Patient registration (name, age, gender)  
- Doctor and specialization selection  
- Automatic fee display based on specialization  
- Appointment confirmation with receipt  
- Simple navigation between screens  

# Technologies Used
- Java  
- Java Swing  
- IntelliJ IDEA  

# How to Run
1. Open the project in IntelliJ IDEA  
2. Run the file `HospitalApp.java`  
3. Follow the steps shown in the interface  

# System Flow
The system starts with entering patient details. After that, the user selects specialization, doctor, and time. Finally, a confirmation screen shows all the details. The user can go back or finish the process.

# Limitations
- The system does not store data permanently  
- No option to save or download the receipt  
- It is a basic version and not connected to any database  



## Future Work
In the future, this project can be improved by adding database support, login system, and receipt download option. More features can be added to make it closer to a real hospital system.
